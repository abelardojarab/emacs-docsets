    DOMString

    In This Article
    1. Specifications
    2. See also

    DOMString is a UTF-16 String. As JavaScript already uses such strings, DOMString is mapped directly to a String.

    Passing null to a method or parameter accepting a DOMString typically stringifies to "null".
