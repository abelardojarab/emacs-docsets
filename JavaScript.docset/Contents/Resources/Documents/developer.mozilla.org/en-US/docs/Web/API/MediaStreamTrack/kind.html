    MediaStreamTrack.kind

    In This Article
    1. Syntax
      1. Value
    2. Example
    3. Specifications
    4. Browser compatibility
    5. See also

    The read-only property MediaStreamTrack.kind returns a DOMString set to "audio" if the track is an audio track and to "video", if it is a video track. It doesn't change if the track is deassociated from its source.

    Syntax

     var type = track.kind;

    Value

    The possible values are a DOMString with on of the following values:

    • "audio": the track is an audio track.
    • "video": the track is a video track.
