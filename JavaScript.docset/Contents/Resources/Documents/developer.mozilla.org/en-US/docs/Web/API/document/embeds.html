    Document.embeds

    In This Article
      1. Summary
      2. Syntax
      3. Specifications

    Summary

    embeds returns a list of the embedded OBJECTS within the current document.

    Syntax

    nodeList = document.embeds

    Specifications

    Specification Status Comment
    WHATWG HTML Living Standard
    The definition of 'Document.embeds' in that specification.    		 Living Standard
