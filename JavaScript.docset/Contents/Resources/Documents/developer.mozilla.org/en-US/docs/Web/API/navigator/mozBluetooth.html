    Navigator.mozBluetooth

    In This Article
    1. Sumário
    2. Sintaxe
    3. Valor
    4. Especificação
    5. Veja também

    Non-standard
    This feature is not on a current W3C standards track, but it is supported on the Firefox OS platform. Although implementations may change in the future and it is not supported widely across browsers, it is suitable for use in code dedicated to Firefox OS apps.

    This API is available on Firefox OS for internal applications only.

    Sumário

    Retorna um objeto BluetoothManager que você pode usar para obter um objeto BluetoothAdapter.

    Sintaxe

    var bluetooth = window.navigator.mozBluetooth;

    Valor

    navigator.mozBluetooth é um objeto BluetoothManager.

    Especificação

    Não é parte de nenhuma especificação.
