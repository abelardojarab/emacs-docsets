    Performance.timing

    In This Article
    1. Syntax
    2. Specifications
    3. Browser compatibility
    4. See also

    The Performance.timing read-only property returns a PerformanceTiming object containing latency-related performance information.

    Syntax

    timingInfo = performance.timing;
