    FileHandle.getFile()

    In This Article
    1. Summary
    2. Syntax
      1. Return
    3. Specifications
    4. See also

    This article needs a technical review. How you can help.

    Non-standard
    This feature is non-standard and is not on a standards track. Do not use it on production sites facing the Web: it will not work for every user. There may also be large incompatibilities between implementations and the behavior may change in the future.

    Summary

    The getFile method allows to retrieve a read-only snapshot of the handled file in the form of a File object.

    Syntax

    var request = instanceOfFileHandle.getFile();

    Return

    A DOMRequest object. In case of success, the request's result is a File object.
