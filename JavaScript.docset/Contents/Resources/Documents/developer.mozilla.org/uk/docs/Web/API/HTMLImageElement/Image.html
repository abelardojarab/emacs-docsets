    Конструктор елемента Image

    Приймає два необов'язкових параметра: Image([unsigned long width, unsigned long height])

    Повертає екземпляр HTMLImageElement так само, як document.createElement('img').

    Приклад:

    var myImage = new Image(100, 200);
myImage.src = 'зразок.jpg';
console.log(myImage);
    Результат:
    <img width="100" height="200" src="зразок.jpg">
    Specification Status Comment
    HTML5
    The definition of the Image constructor in that specification.    		 Recommendation Додано конструктор (з двома необов'язковими параметрами).
    Наступні властивості відтепер застарілі: name, border, align, hspace, vspace, та longdesc.
    Властивості height та width відтепер мають тип unsigned long замість long.
    Додано властивість crossorigin.
